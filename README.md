# Ford-Fulkerson-Algorithm
# Hackathon Network Optimization 🌐

A final project for the **Operations Research** course. This project applies the **Ford-Fulkerson Algorithm** to optimize the Wi-Fi and data network infrastructure of a student hackathon.

## 🎯 Project Goals
* **Maximum Flow:** Calculate the maximum data bandwidth the network can handle.
* **Minimum Cut (Bottlenecks):** Identify the exact cables/routers that limit the network's speed, showing where IT budgets should be invested.

## 💻 Technologies & Files
* **`app.py`**: An interactive web app built with **Python (Streamlit, NetworkX, Matplotlib)** that visualizes the algorithm step-by-step.
* **`main.cpp`**: The core algorithmic logic implemented in **C++**.
